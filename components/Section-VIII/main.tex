\section{\textbf{KẾT LUẬN VÀ HƯỚNG PHÁT TRIỂN}}

\subsection{Kết luận}

Đồ án "VeritaShop" đã hoàn thành mục tiêu xây dựng ứng dụng thương mại điện tử chuyên biệt cho điện thoại:

\textbf{Kết quả đạt được:}
\begin{itemize}
    \item Ứng dụng Flutter đa nền tảng (Android, iOS, Web), Backend Node.js/Express.js ổn định
    \item Tích hợp MoMo payment (Deep Link + IPN), Cloudinary CDN
    \item \textbf{ABSA với VisoBERT-STL}: AD F1=89.39\%, SC F1=96.37\% (dataset 14,912 bình luận)
    \item Content Moderation tự động phát hiện nội dung vi phạm
    \item Admin Dashboard quản lý toàn diện sản phẩm, đơn hàng, người dùng
\end{itemize}

\textbf{Hạn chế:} Chưa có Push Notifications, đa ngôn ngữ, VNPay/ZaloPay, chat support, MoMo chưa production.

\subsection{Hướng phát triển}

\begin{itemize}
    \item \textbf{Tính năng}: Push notification (FCM), social login, so sánh sản phẩm, đa ngôn ngữ
    \item \textbf{Thanh toán}: VNPay, ZaloPay, Visa/Mastercard, trả góp 0\%
    \item \textbf{Trải nghiệm}: Real-time chat, order tracking, AR preview
    \item \textbf{AI/ML}: Product recommendation, chatbot AI (Gemini), image search
    \item \textbf{Scale}: Redis caching, microservices, CI/CD pipeline
\end{itemize}

Đồ án minh chứng khả năng ứng dụng công nghệ hiện đại (Flutter, Node.js, MongoDB, BERT) vào thương mại điện tử, đặc biệt tích hợp ABSA đạt hiệu quả cao mở hướng nghiên cứu NLP tại Việt Nam.
