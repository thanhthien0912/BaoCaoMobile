\begin{itemize}
    \item Đăng nhập/đăng ký với JWT, PIN Lock bảo mật
    \item Danh sách sản phẩm với search/filter/sort, chi tiết với specs đầy đủ
    \item Giỏ hàng, wishlist, checkout 5 bước
    \item Thanh toán MoMo với Deep Link + IPN callback
    \item Đánh giá sản phẩm với ABSA analysis
    \item Lịch sử đơn hàng với tracking trạng thái
\end{itemize}
